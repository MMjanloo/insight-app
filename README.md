# VeChain Insight
Serverless VeChain Explorer.

It's deployed at https://vechain.github.io/insight.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
